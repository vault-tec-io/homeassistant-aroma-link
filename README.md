e# Aroma-Link Diffuser Integration for Home Assistant

Control your Aroma-Link diffuser through Home Assistant.

## Features
- Power on/off control
- Fan control
- Work/Pause phase monitoring
- Schedule configuration
- Real-time status updates

## Installation
1. Copy the `aromalink` folder to your `custom_components` directory
2. Restart Home Assistant
3. Add integration through UI (Settings -> Devices & Services -> Add Integration)
4. Enter your Aroma-Link credentials

## Entities
- **Switch**: Power and fan control
- **Sensor**: Phase status and countdown timers
- **Number**: Work and pause duration settings
